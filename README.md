An advanced, modern, and production-ready **Hospital Management System** designed to streamline clinical workflows, patient management, and professional billing. This project has been significantly overhauled and taken to the **Next Level** with premium UI/UX, robust security, and high-quality clinical report generation.

---

## 🚀 "Next Level" Evolution
This repository represents a major leap from the legacy version, introducing industry-standard features and a professional "Dark-Structural" design system.

### 💎 Premium UI/UX & Design
- **Unified Auth Portal**: A centralized, sleek authentication interface on the homepage with tabbed access for Patients, Doctors, and Receptionists.
- **"Dark-Structural" Branding**: A curated deep navy (`#102742`) and airy blue (`#f2f6fb`) design system for a trustworthy, institutional feel.
- **Responsive Dashboard**: Fully modernized sidebars with smooth micro-animations and mobile-first layouts.
- **Glassmorphism**: Elegant transparent elements and subtle shadows for a state-of-the-art interface.

### 📄 Professional Clinical PDF Generation
- **Full-Bleed Branding**: Edge-to-edge institutional headers and footers for all prescriptions and bills.
- **Symmetrical Layout**: Perfectly balanced 5mm side margins for professional printing alignment.
- **Single-Page Optimization**: Intelligent layout logic ensuring clinical reports fit on a single A4 page.
- **Digital Signatures**: Authorized signature placeholders and clinical summary tables.

### 🛡️ Security & Performance
- **CSRF Protection**: Integrated Cross-Site Request Forgery protection across all critical forms.
- **Input Sanitization**: Robust backend handling of all user-submitted data.
- **Audit Logging**: Integrated clinical audit logging for tracking key system actions.
- **Modern File Structure**: Descriptive, industry-standard filenames (e.g., `patient-auth.php` instead of `func.php`) for easier maintainability.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript, Bootstrap 4.
- **Backend**: PHP 8.x (Modernized Controller Logic).
- **Database**: MySQL (Optimized Relational Schema).
- **PDF Engine**: TCPDF (Customized for Premium Branding).
- **Security**: CSRF Tokenization & SQL Injection Prevention.

## 🔧 Installation & Setup

1. **Environment**: Install [XAMPP](https://www.apachefriends.org/) or any LAMP/WAMP stack.
2. **Clone**: `git clone https://github.com/kishan0725/Hospital-Management-System.git`
3. **Move**: Place the project folder inside your `htdocs` directory.
4. **Database**:
   - Open `phpMyAdmin` and create a database named `myhmsdb`.
   - Import the `myhmsdb.sql` file provided in the root directory.
5. **Config**: Ensure `include/config.php` matches your local DB credentials.
6. **Launch**: Navigate to `http://localhost/Hospital-Management-System` in your browser.
